For the Robots the decimal format is not convenient. And if they asked to count "1" in a number, then they want to count it in the binary representation of the number. You can read more about binary here.

You are given a number (a positive integer). You should convert it to the binary format and count how many unities in it.

Hints: This task can be easily solved with two functions -- {% if "python 2" in interpreter|lower %} bin and count. {% else %} bin and count. {% endif %}

Input: A number as a positive integer.

Output: The quantity of unities in the binary form, an integer.

Example: 

checkio(4) == 1
checkio(15) == 4
checkio(1) == 1
checkio(1022) == 9
